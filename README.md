
  # AI Fridge Web App: Groceries Tracker

  Pitching Video: https://drive.google.com/file/d/1_20SLVKQtpfTJRiu2ncwqvZXF2vmkZ7f/view?usp=sharing

  Trello: https://trello.com/b/JoE7K1FA/ai-fridge

An AI-powered web application that helps users manage their fridge inventory, reduce food waste, and track available ingredients efficiently.

This project was inspired by a hackathon design and focuses on improving household food management through a simple and interactive interface.

---

## 🚀 Features

- 📦 Add and manage fridge items  
- 🗂️ AI-based recipt reader  
- ⏳ Track expiration dates  
- 🔍 Search and filter items easily  
- 🤖 AI-based recipe suggestions   

---

## 🛠️ Tech Stack

- **Frontend:** React + Vite  
- **UI Libraries:** Material UI, Radix UI  
- **Styling:** Tailwind CSS  
- **Other Tools:** React Hook Form, Recharts, Framer Motion  

---


  ## 📦 Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
